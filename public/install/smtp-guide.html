SMTP Guide — Gmail (App Password)

A compact, installer-friendly HTML doc that walks users through enabling 2‑Step Verification, creating a Google App Password, and the exact SMTP settings and example code they can copy into the app.

Quick checklist

  • Enable 2‑Step Verification on your Google Account.
  • Create an App Password and copy the 16‑character string.
  • Use smtp.gmail.com with TLS (port 587) or SSL (port 465).
  • Enter your full Gmail address as username and the App Password as password in the installer.

Step 1 — Enable 2‑Step Verification

1

Open myaccount.google.com/security → Signing in to Google → 2‑Step Verification. Follow the on‑screen steps (Phone or Authenticator app).

2

Once 2‑Step is active, you can create App Passwords.

Step 2 — Create an App Password

From myaccount.google.com/security → App passwords. Pick Mail as app and your device or "Other (custom name)". Click Generate and copy the 16‑character password (no spaces when used).

SMTP settings

Setting Value
SMTP Host smtp.gmail.com
SMTP Port (TLS) 587
SMTP Port (SSL) 465
Encryption TLS (STARTTLS) or SSL
Username Your full Gmail address (example@gmail.com)
Password Your 16‑character Google App Password
Authentication Required (Yes)

Tip: use port 587 with STARTTLS for best compatibility.

Example — PHP (PHPMailer)

use PHPMailer\PHPMailer\PHPMailer;
use PHPMailer\PHPMailer\Exception;

$mail = new PHPMailer(true);
try {
  $mail->isSMTP();
  $mail->Host = 'smtp.gmail.com';
  $mail->SMTPAuth = true;
  $mail->Username = 'your_email@gmail.com';
  $mail->Password = 'your_16_char_app_password';
  $mail->SMTPSecure = PHPMailer::ENCRYPTION_STARTTLS;
  $mail->Port = 587;

  $mail->setFrom('your_email@gmail.com', 'Your Name');
  $mail->addAddress('recipient@example.com');
  $mail->Subject = 'Test email';
  $mail->Body    = 'Hello from PHPMailer via Gmail App Password';

  $mail->send();
  echo 'Message sent';
} catch (Exception $e) {
  echo "Mailer Error: {$mail->ErrorInfo}";
}

Example — Node (Nodemailer)

const nodemailer = require('nodemailer');

let transporter = nodemailer.createTransport({
  host: 'smtp.gmail.com',
  port: 587,
  secure: false, // use STARTTLS
  auth: {
    user: 'your_email@gmail.com',
    pass: 'your_16_char_app_password'
  }
});

await transporter.sendMail({
  from: 'Your Name ',
  to: 'recipient@example.com',
  subject: 'Test email',
  text: 'Hello from Nodemailer via Gmail App Password'
});

Example — Python (smtplib)

import smtplib
from email.message import EmailMessage

msg = EmailMessage()
msg.set_content('Hello from Python')
msg['Subject'] = 'Test email'
msg['From'] = 'your_email@gmail.com'
msg['To'] = 'recipient@example.com'

with smtplib.SMTP('smtp.gmail.com', 587) as s:
    s.ehlo()
    s.starttls()
    s.login('your_email@gmail.com', 'your_16_char_app_password')
    s.send_message(msg)

Troubleshooting

  • If login fails: confirm 2‑Step Verification is enabled and you're using the App Password (not your normal Gmail password).
  • For Google Workspace accounts, admin may disable App Passwords — contact your admin.
  • Watch SMTP limits: regular Gmail accounts have daily sending limits (~500 messages/day).
  • If connections time out, ensure the server where your app runs allows outbound SMTP to ports 587 or 465 (cloud hosts sometimes block them).
